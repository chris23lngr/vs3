# Security Policy

## Supported Versions

VS3 supports the latest major version and the previous major version. Security fixes are backported when feasible.

## Reporting a Vulnerability

Please report security issues privately.

Preferred method:

1. Open a GitHub Security Advisory for this repository.
1. Provide steps to reproduce, impact assessment, and any proof-of-concept details.

What to expect:

- Acknowledgement within 72 hours.
- Status updates as we triage and fix the issue.
- Coordinated disclosure once a fix is available.

Please avoid filing public issues for security reports.
